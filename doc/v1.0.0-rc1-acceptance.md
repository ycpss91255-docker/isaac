## 3. 已知、刻意的 scope reduction（給 H4 判斷用，**別當缺陷**）

- **SimulationContext 仍 deferred**（#151 `reset()` hang driver shutdown）：driver/example 走 `AppLauncher` + `omni.timeline` loop；L1-L4 + cmd_vel 不需 physics stepping。→ **是否在 v1.0.0 前要補，是 H4。**
- **`prop_cube` 物理是結構性 authored**（有 `RigidBodyAPI`），但無 sim stepping，不會真的掉落/被推。隨 SimulationContext 一起 deferred。
- **`test_l1_urdf_to_usd_diff_zero` 退役**（契約由 `test_model_import.py` 覆蓋）。
- **committed `camera_bot.usd`** = legacy importer + `defaultPrim` patch（usd-core，免 GPU），非新 importer 重生。
- **lidar / imu = `NotImplementedError`**：刻意 out-of-scope callout。
- **Isaac Lab pin `v2.3.0`**（非最新 2.3.x，importer skew，見 Dockerfile 註解）。
